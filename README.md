## Developing / Building Locally

1.  Clone this repo
2.  Install the dependencies using [yarn](https://yarnpkg.com/lang/en/docs/install/):

```
yarn install
```

3.  Build the app:

```
yarn build
```

4.  Then, open the repo's [index.html](index.html) in your browser and enjoy!
